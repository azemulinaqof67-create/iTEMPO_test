# Справочник кабинетов АО «КМК "ТЭМПО"» (АБК-2)

Все указанные подразделения АО «КМК "ТЭМПО"» располагаются в здании **АБК-2**.

| Кабинеты | Подразделение |
|----------|---------------|
| 201 | Приемная |
| 204, 217 | Финансово-экономический отдел |
| 205 | Отдел труда и заработной платы (ОТиЗ) |
| 207, 209 | Отдел продаж КМК, Отдел закупок основных материалов КМК, Отдел анализа рынка, Отдел продаж КПК |
| 208 | Коммерческий директор |
| 210 | Отдел снабжения |
| 218 | Отдел логистики и закупки ПШМ |
| 219 | Бухгалтерия |
| 220 | Архив |
| 222 | Юридический отдел |
