# Документы для трудоустройства в ГК «ТЭМПО»

**Телефон отдела кадров:** 8-800-775-20-21

> Прием документов осуществляется строго **по записи с 8:00 до 11:00**

---

## Основные документы

| Документ |
|----------|
| Паспорт |
| ИНН |
| СНИЛС |
| Трудовая книжка или справка по форме СТД-Р от предыдущего работодателя |
| Документы об окончании учебных заведений, подтверждающие квалификацию |
| Военный билет, приписное свидетельство или справка из военкомата |
| Фотографии 2 штуки размером 3х4 |

## Медицинские требования

### В поликлинике по месту жительства:
- Справка о прививке **АДСМ** (от столбняка)
- Результат **флюорографии** (срок действия — 1 год)

### Для иногородних:
- Справка из **ПНД** о том, что вы не состоите на учете
- Справка из **ГНД** о том, что вы не состоите на учете
