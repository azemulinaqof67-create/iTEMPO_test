# Приказ № 013/1537

**О корпоративном стиле одежды**

| Параметр | Значение |
|----------|----------|
| **Организация** | ГК «ТЭМПО» |
| **Номер** | 013/1537 |
| **Дата** | 30 августа 2013 г. |
| **Город** | Набережные Челны |
| **Подписант** | Генеральный директор Ф.Ф. Шарифуллин |
| **Дата вступления** | 02.09.2013 г. |

## Текст приказа

В связи с переходом на осеннее время года, в целях соблюдения внутренней дисциплины труда, а также введением корпоративного стиля организации

**ПРИКАЗЫВАЮ:**

1. Со 02.09.2013г. ввести требование к корпоративному стилю одежды сотрудников ГК «ТЭМПО», согласно Приложению №1.
2. Руководителям структурных подразделений ознакомить своих работников с приказом.
3. Контроль и ответственность за соблюдение требований данного приказа возложить на руководителей структурных подразделений.
